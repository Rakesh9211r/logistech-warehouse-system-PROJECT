# Logistech Warehouse System

A comprehensive warehouse management system with intelligent algorithms for bin selection, cargo loading, and conveyor system optimization.

## Features

- **Intelligent Bin Selection**: First-fit, best-fit, and priority-based algorithms
- **Cargo Loading Optimization**: Knapsack algorithm for efficient loading
- **Conveyor System**: Priority-based package scheduling
- **Database Integration**: SQLite database for data persistence
- **Singleton Pattern**: Shared resource management
